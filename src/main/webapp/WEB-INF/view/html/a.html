我校召开汉语言文学专业认证指导专家意见反馈会

5月14日下午，四川师范大学汉语言文学专业认证指导专家意见反馈会在狮子山校区7教105会议室召开。以华中师范大学原副校长李向农为组长的认证指导专家组全体成员，校党委常委、副校长杜伟出席会议，学校主要职能部门负责人，文学院行政教师及汉语言文学专业全体教师，师范专业举办学院院长、教学副院长、专业负责人参加会议。会议由李向农主持。 李向农介绍了评估工作的开展情况，对实地考察学校教师教育相关的教学条件设施，召开教师学生座谈会，抽查专业课程考试试卷、毕业论文、实习实践资料等情况进行了说明，并代表专家组反馈审核评估考察意见，在肯定学校及专业办学成绩的基础上，从持续改进的角度对我校加强专业教学工作提出有关意见和建议。随后，专家组成员四川师范大学原副校长祁晓玲、陕西师范大学教育学院副院长吴合文、福建师范大学文学院副院长林志强、淮阴师范学院教学质量监控与评估中心主任胡元林先后对我校汉语言文学认证中出现的问题提出了一些建议和意见。 文学院院长刘敏在发言中感谢各位专家反馈的宝贵意见和建议，让学院行政、老师发现了专业准备上的不足，有助于学院进一步明确下一步整改的方向，并标准将认真整改。以饱满的热情迎接教育部专家的正式认证。 杜伟首先对各位专家和老师的辛勤劳作和真诚的指导表达了衷心的感谢。杜伟谈到专家们通过多种方式严格按照教育部师范专业认证的程序和方式对我校汉语言文学专业进行了专业细致地考察，指出专家们的评价判断特别精准，问题分析深入，整改建议特别到位，为我校汉语言文学专业在正式接受认证前接受了一次高端的问诊和精准的把脉，问题建议既一针见血又对症下药，对我们的问题和失误提出了很好的指引，并表示对于专家们提出的问题一定认真解决，对于专家们提出的建议强调一方面文学院要针对问题制定出切实可行的整改方案，建立任务清单、职能清单，另一方面各单位要进一步加强质量文化建设，人才培养质量是高校建设培养的生命线，要让质量建设上升到制度和文化的层面，让其成为推动学校事业发展的强大动力。
